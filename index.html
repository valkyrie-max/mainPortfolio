Laark

A virtual gallery application that allows users to look at art.

HTML | SCSS | JQuery | REST API | Paired programming

The main task of the project was to work in a pair and practice the strong type of paired programming while communicating in an effective way. Futhermore, the web application had to be interactive, accessible and fully responsive.

In this application, the user starts by selecting a theme. Then, the app presents three paintings to pick from. Based on the painting the user choose, two more related art pieces are exhibited.

No doubt, this project helped me to enhance my human skills by working together with the teammate. I also got a chance to continue to improve my tech skills by introducing myself to namespacing to ensure the readability of code.

Frame of mind

A place where you can let it all out.

SCSS | JSX | REACT | JavaScript | Firebase

The primary objective of the project was to learn more about React.js library by creating an app that could involve API and/or Firebase.It also included a good chunk of error handling and more practice with UI/UX.

Frame of Mind is a place where the user can get everything off their chest. They start by choosing an emotion from the list. Then, they can either listen to the offered melody or not while writing about what they are feeling in the moment. After the user is finished, they can share their thoughts with others.

This is one of the projects I am really proud of. It taught me the importance of MVP, enhanced awareness of error handling and UX.Additionally, this project was an essential tool for me to understand and get a good grip of React and Firebase basics.

BattleRockets

Wreck the enemy's rockets before they destroy yours.

SCSS | JSX | REACT | JavaScript | API | Mob programming

This project's goal was to mimic the agency-like environment where as a team of 3-4 developers each team had to create a web application based on the client's brief. The project was focused on improving team working skills as well as React, JavaScript, and many other tech skills.

BattleRockets is a two-player game similar to classic battleship but with rockets instead. The objective of the application is to educate players about SpaceX rockets. By hitting a rocket, the user gets a popup with info about one.

Just like the previous project, I am proud of this one. It helped me to drastically improve my human skills get a deeper understanding of React and REST API. The project was created along side with Mariya Morosovska, Miguel Cabrera and Lucas Silbernagel.
